# Zenith Boutique | OS Master Engine v4.5 💎

Zenith Boutique is a high-end, realistic fashion styling engine built on the custom **Zenith Kernel**. Designed for high performance and stability, it features a modular architecture inspired by operating system kernels.

## ✨ Key Features
- **Master Kernel v4.5:** A centralized orchestrator that manages data, UI, rendering, and game logic.
- **Stochastic Blink Engine:** Realistic, randomized eye-blinking for a life-like doll appearance.
- **Auto-Healing Watchdog:** Detects logic glitches and memory corruption, automatically recovering the safe state.
- **Cinematic Camera:** Fluid zooming modes for Face, Dress, and Full-Body views optimized for 120Hz displays.
- **Gems Economy:** Fully functional shop system with Lucky Spins and Anti-Cheat daily rewards.
- **Persistence Layer:** Automatically saves unlocked items, gems, and fashion history to LocalStorage.

## 🚀 Technical Highlights
- **Layered Rendering Stack:** Efficient canvas drawing pipeline.
- **Delta Clamping:** Physics-based animations that remain stable even during frame drops.
- **Mobile Optimized:** Full hardware acceleration support for devices like the **Nothing Phone (3a)**.

## 🛠️ How to Run
1. Download the `index.html` file.
2. Open it in any modern web browser.
3. To host it, simply push the file to a GitHub repository and enable **GitHub Pages**.
